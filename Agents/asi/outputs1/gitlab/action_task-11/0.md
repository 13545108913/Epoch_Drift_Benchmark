## Reusable Functions
```python
def navigate_to_commits(project_link_id: str, commits_link_id: str):
    """Navigate from project page to commits page.
    
    Args:
        project_link_id: The ID of the project link
        commits_link_id: The ID of the commits link
        
    Returns:
        None
        
    Examples:
        navigate_to_commits('320', '266')
    """
    click(project_link_id)  # Click project link to navigate to project page
    click(commits_link_id)  # Click commits link to access commit history
```

```python
def scroll_commits_page(scroll_amounts: list):
    """Scroll through commits page to find specific dates.
    
    Args:
        scroll_amounts: List of vertical scroll amounts (pixels)
        
    Returns:
        None
        
    Examples:
        scroll_commits_page([500, 800, 1000])
    """
    for amount in scroll_amounts:
        scroll(0, amount)  # Scroll down by specified amount
```
