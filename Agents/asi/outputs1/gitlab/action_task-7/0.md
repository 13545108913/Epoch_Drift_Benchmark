## Reusable Functions
```python
def navigate_to_repo_commits(repo_link_id: str, commits_link_id: str):
    """Navigate from a repository page to its main commits page.
    
    Args:
        repo_link_id: The ID of the repository link
        commits_link_id: The ID of the "Commits" link in the repository section
        
    Returns:
        None
        
    Examples:
        navigate_to_repo_commits('286', '265')
    """
    click(repo_link_id)  # Click repository link to access the repo
    click(commits_link_id)  # Click commits link to access commit history
```

```python
def switch_to_main_branch(branch_selector_id: str, main_branch_id: str):
    """Switch from current branch to main branch in repository view.
    
    Args:
        branch_selector_id: The ID of the branch selector dropdown
        main_branch_id: The ID of the main branch option
        
    Returns:
        None
        
    Examples:
        switch_to_main_branch('473', '1462')
    """
    click(branch_selector_id)  # Click branch selector to open dropdown
    click(main_branch_id)  # Click main branch option to switch
```
