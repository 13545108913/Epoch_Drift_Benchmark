## Reusable Functions
```python
def navigate_to_commit_history(repo_link_id: str):
    """Navigate to the commit history page of a repository.
    
    Args:
        repo_link_id: The ID of the repository link
        
    Returns:
        None
        
    Examples:
        navigate_to_commit_history('286')
    """
    click(repo_link_id)  # Click repository link
    click('653')  # Click History link to access commit history
```

```python
def switch_branch(branch_button_id: str, target_branch_id: str):
    """Switch to a different branch in the repository.
    
    Args:
        branch_button_id: The ID of the branch dropdown button
        target_branch_id: The ID of the target branch in the dropdown
        
    Returns:
        None
        
    Examples:
        switch_branch('473', '1462')
    """
    click(branch_button_id)  # Click branch dropdown button
    click(target_branch_id)  # Click target branch to switch
```
