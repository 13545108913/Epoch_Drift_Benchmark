## Reusable Functions
```python
def navigate_to_profile_settings(profile_link_id: str):
    """Navigate to profile settings page.
    
    Args:
        profile_link_id: The ID of the profile link/button
        
    Returns:
        None
        
    Examples:
        navigate_to_profile_settings('275')
    """
    click(profile_link_id)  # Click profile link to access settings
    keyboard_press("Escape")  # Close any dropdown overlays if present
    click(profile_link_id)  # Click profile link again to ensure navigation
```

```python
def update_profile_url(url_field_id: str, update_button_id: str, url: str):
    """Update website URL in profile settings.
    
    Args:
        url_field_id: The ID of the URL input field
        update_button_id: The ID of the update/save button
        url: The URL to set as homepage
        
    Returns:
        None
        
    Examples:
        update_profile_url('878', '914', 'byteblaze.github.io')
    """
    fill(url_field_id, url)  # Fill the URL field with the provided URL
    click(update_button_id)  # Click update button to save changes
```
