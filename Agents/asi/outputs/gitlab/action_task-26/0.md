## Reusable Functions
```python
def navigate_to_access_tokens(avatar_id: str, preferences_id: str, tokens_id: str):
    """Navigate from homepage to access tokens page.
    
    Args:
        avatar_id: The ID of the user avatar/link
        preferences_id: The ID of the "Preferences" link
        tokens_id: The ID of the "Access Tokens" link
        
    Returns:
        None
        
    Examples:
        navigate_to_access_tokens('225', '241', '314')
    """
    click(avatar_id)  # Click user avatar to open dropdown
    click(preferences_id)  # Click Preferences in dropdown
    click(tokens_id)  # Click Access Tokens link
```

```python
def get_rss_token():
    """Retrieve and return the RSS feed token from the page.
    
    Args:
        None
        
    Returns:
        str: The RSS feed token value
        
    Examples:
        token = get_rss_token()
    """
    token_element = page.get_by_text("Personal Access Tokens").locator("..").get_by_role("code")
    token_value = token_element.inner_text()
    return token_value
```
