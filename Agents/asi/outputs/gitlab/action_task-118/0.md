## Reusable Functions
```python
def navigate_to_milestones(issues_id: str, milestones_id: str):
    """Navigate from the main page to the milestones creation page.
    
    Args:
        issues_id: The ID of the Issues link
        milestones_id: The ID of the Milestones link
        
    Returns:
        None
        
    Examples:
        navigate_to_milestones('347', '371')
    """
    click(issues_id)  # Click Issues link
    click(milestones_id)  # Click Milestones link
```

```python
def create_milestone(new_milestone_id: str, title_id: str, title: str, start_date_id: str, start_date: str, due_date_id: str, due_date: str, create_button_id: str):
    """Fill out and submit the milestone creation form.
    
    Args:
        new_milestone_id: The ID of the "New milestone" link
        title_id: The ID of the Title field
        title: The milestone title
        start_date_id: The ID of the Start Date field
        start_date: The start date in YYYY-MM-DD format
        due_date_id: The ID of the Due Date field
        due_date: The due date in YYYY-MM-DD format
        create_button_id: The ID of the "Create milestone" button
        
    Returns:
        None
        
    Examples:
        create_milestone('559', '551', 'Project Launch', '626', '2023-05-01', '633', '2023-05-15', '637')
    """
    click(new_milestone_id)  # Click New milestone link
    fill(title_id, title)  # Fill milestone title
    fill(start_date_id, start_date)  # Fill start date
    fill(due_date_id, due_date)  # Fill due date
    click(create_button_id)  # Click Create milestone button
```
