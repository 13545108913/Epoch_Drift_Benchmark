## Reusable Functions

```python
def star_repository(repo_link_id: str, star_button_id: str):
    """Navigate to a repository page and star it.
    
    Args:
        repo_link_id: The ID of the repository link element
        star_button_id: The ID of the star button element
        
    Returns:
        None
        
    Examples:
        star_repository('777', '663')
        star_repository('742', '624')
    """
    click(repo_link_id)  # Click on repository link to navigate to its page
    click(star_button_id)  # Click star button to star the repository
    go_back()  # Navigate back to repository list
```

```python
def star_multiple_repositories(repo_star_pairs: list):
    """Star multiple repositories by providing their link and button IDs.
    
    Args:
        repo_star_pairs: List of tuples containing (repo_link_id, star_button_id)
        
    Returns:
        None
        
    Examples:
        star_multiple_repositories([('777', '663'), ('742', '624'), ('710', '617')])
    """
    for repo_link_id, star_button_id in repo_star_pairs:
        click(repo_link_id)  # Click on repository link
        click(star_button_id)  # Click star button
        go_back()  # Navigate back to list
```
