## Reusable Functions
```python
def navigate_to_contributors(analytics_id: str, repository_id: str, contributors_id: str):
    """Navigate from homepage to contributors statistics page.
    
    Args:
        analytics_id: The ID of the "Analytics" link
        repository_id: The ID of the "Repository" link
        contributors_id: The ID of the "Contributors" link
        
    Returns:
        None
        
    Examples:
        navigate_to_contributors('417', '416', '530')
    """
    click(analytics_id)  # Click Analytics link
    click(repository_id)  # Click Repository link
    click(contributors_id)  # Click Contributors link
```

```python
def switch_branch_and_get_contributor_stats(branch_selector_id: str, branch_name: str):
    """Switch to a specific branch and extract contributor statistics.
    
    Args:
        branch_selector_id: The ID of the branch selector dropdown
        branch_name: The name of the branch to switch to
        
    Returns:
        None
        
    Examples:
        switch_branch_and_get_contributor_stats('507', 'main')
    """
    click(branch_selector_id)  # Click branch selector dropdown
    fill(branch_selector_id, branch_name)  # Enter branch name
    keyboard_press("Enter")  # Press Enter to switch branch
    # Contributor statistics are now displayed for the selected branch
```
