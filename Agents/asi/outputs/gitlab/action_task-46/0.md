## Reusable Functions
```python
def navigate_to_repo_analytics(repo_id: str, analytics_id: str):
    """Navigate to the analytics page of a repository.
    
    Args:
        repo_id: The ID of the repository link
        analytics_id: The ID of the Analytics link
        
    Returns:
        None
        
    Examples:
        navigate_to_repo_analytics('385', '478')
    """
    click(repo_id)  # Click on the repository
    click(analytics_id)  # Click on Analytics link


def navigate_to_contributors_page(repo_analytics_id: str, contributors_id: str):
    """Navigate from repository analytics to contributors page.
    
    Args:
        repo_analytics_id: The ID of the Repository link in analytics
        contributors_id: The ID of the Contributors link
        
    Returns:
        None
        
    Examples:
        navigate_to_contributors_page('479', '339')
    """
    click(repo_analytics_id)  # Click Repository link in analytics
    click(contributors_id)  # Click Contributors link
```
