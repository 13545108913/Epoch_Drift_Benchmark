运行 GitLab 服务
1. 启动服务
bash
# 在包含 docker-compose.v15.yml 文件的目录中执行
docker-compose -f docker-compose.v15.yml up -d

2. 查看服务状态
bash
# 查看容器运行状态
docker-compose -f docker-compose.v15.yml ps

# 查看实时日志
docker-compose -f docker-compose.v15.yml logs -f


停止 GitLab 服务
1. 停止服务（保留数据）
bash
# 停止运行中的服务
docker-compose -f docker-compose.v15.yml stop

2. 停止并删除容器（保留数据）
bash
# 停止并删除容器，但保留卷数据
docker-compose -f docker-compose.v15.yml down

3. 完全停止并清理所有资源
bash
# 停止并删除容器、网络，但保留数据卷
docker-compose -f docker-compose.v15.yml down --volumes
# 注意：--volumes 会删除数据，请谨慎使用！
